# NextHire — The AI Recruiting Brain

Full **Next.js + FastAPI** application over the *Council-of-Nine* candidate
ranking engine: explainable scoring, integrity (honeypot) screening, a fairness
audit (EU AI Act, 4/5ths rule), analytics and Excel export — CPU-only and
offline.

This Space runs the prebuilt container image published to GHCR by the project's
GitHub Actions workflow. Upload your own JSON/JSONL candidate pool, pick a role,
tune the Council weights, and rank.

- Image: `ghcr.io/prajwalbr0304/nexthire:latest`
